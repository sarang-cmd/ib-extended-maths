## 1. Quadratic function \( y = x^2 - 4x + 3 \)

**(a) Complete the table**

| \(x\) | \(y\) |
|------|------|
| \(0\) | \(3\) |
| \(1\) | \(0\) |
| \(2\) | \(-1\) |
| \(3\) | \(0\) |
| \(4\) | \(3\) |
| \(5\) | \(8\) |

**(b) Intersection points with the axes**  

- \(y\)-intercept: \((0,3)\)  
- \(x\)-intercepts:
  \[
  x^2 - 4x + 3 = 0
  \]
  \[
  (x-1)(x-3)=0
  \]
  Points: \((1,0)\) and \((3,0)\)

**(c) Vertex**  
\[
x_v = -\frac{-4}{2\cdot1} = 2
\]
\[
y_v = 2^2 - 4(2) + 3 = -1
\]
Vertex: \((2,-1)\)

**(d) Concavity**  
Concave up.

## 2. Quadratic function \( y = -\frac{1}{2}x^2 - x + 4 \)

**(a) Complete the table**

| \(x\) | \(y\) |
|------|------|
| \(-4\) | \(0\) |
| \(-2\) | \(4\) |
| \(0\) | \(4\) |
| \(1\) | \(\frac{5}{2}\) |
| \(2\) | \(0\) |
| \(4\) | \(-8\) |

**(b) Intersection points with the axes**  

- \(y\)-intercept: \((0,4)\)  
- \(x\)-intercepts:
  \[
  -\frac{1}{2}x^2 - x + 4 = 0
  \]
  Multiply by \(-2\):
  \[
  x^2 + 2x - 8 = 0
  \]
  \[
  (x+4)(x-2)=0
  \]
  Points: \((-4,0)\) and \((2,0)\)

**(c) Vertex**  
\[
x_v = -\frac{-1}{2(-\frac{1}{2})} = -1
\]
\[
y_v = -\frac{1}{2}(-1)^2 - (-1) + 4 = -\frac{1}{2} + 1 + 4 = \frac{9}{2}
\]
Vertex: \(\left(-1, \frac{9}{2}\right)\)

**(d) Concavity**  
Concave down.

**(f) Intersection with \( y = -x + 1 \)**  
\[
-\frac{1}{2}x^2 - x + 4 = -x + 1
\]
\[
-\frac{1}{2}x^2 + 4 = 1
\]
\[
-\frac{1}{2}x^2 = -3
\]
\[
x^2 = 6
\]
\[
x = \pm\sqrt{6}
\]

Using \(y=-x+1\):

For \(x=\sqrt{6}\):
\[
y = 1 - \sqrt{6}
\]

For \(x=-\sqrt{6}\):
\[
y = 1 + \sqrt{6}
\]

Points:
\[
(\sqrt{6}, 1-\sqrt{6})
\]
and
\[
(-\sqrt{6}, 1+\sqrt{6})
\]

## 5. Find the equation of a quadratic with one \(x\)-intercept at \((-1,0)\) and vertex \((1,12)\)

By symmetry, the other \(x\)-intercept is \(x=3\).

\[
y = a(x+1)(x-3)
\]

Use the vertex \((1,12)\):
\[
12 = a(1+1)(1-3) = a(2)(-2) = -4a
\]
\[
a=-3
\]

\[
y = -3(x+1)(x-3)
\]
\[
y = -3(x^2 - 2x - 3) = -3x^2 + 6x + 9
\]

## 6. Quadratic function \( y = x^2 + bx + c \)

**(a)** \(y\)-intercept \((0,6)\) gives
\[
c=6
\]

**(b)** Passes through \((2,0)\):
\[
0 = (2)^2 + 2b + 6
\]
\[
0 = 4 + 2b + 6
\]
\[
0 = 10 + 2b
\]
\[
b=-5
\]

**(c)** Full equation:
\[
y = x^2 - 5x + 6
\]

**(d) Range**  
\[
x_v = -\frac{-5}{2} = \frac{5}{2}
\]
\[
y_v = \left(\frac{5}{2}\right)^2 - 5\left(\frac{5}{2}\right) + 6
= \frac{25}{4} - \frac{25}{2} + 6
= -\frac{1}{4}
\]
Range:
\[
\left[-\frac{1}{4}, \infty\right)
\]

## 7. Find the intersection points of  
\( f(x)=x^2+x-2 \) and \( g(x)=3x-2 \)

Set equal:
\[
x^2 + x - 2 = 3x - 2
\]
\[
x^2 - 2x = 0
\]
\[
x(x-2)=0
\]
\[
x=0 \quad \text{or} \quad x=2
\]

For \(x=0\):
\[
y = 3(0)-2 = -2
\]
Point: \((0,-2)\)

For \(x=2\):
\[
y = 3(2)-2 = 4
\]
Point: \((2,4)\)

## 8. Functions \( f(x)=-2x^2+bx+1 \) and \( g(x)=x+1 \)

**(a)** Since one intersection occurs at \(x=0\):
\[
f(0)=g(0)
\]
\[
1=1
\]
This is true for any \(b\), so \(b\) is not fixed.

From
\[
-2x^2 + bx + 1 = x + 1
\]
\[
-2x^2 + (b-1)x = 0
\]
\[
x[-2x + (b-1)] = 0
\]
The second intersection has
\[
x = \frac{b-1}{2}
\]

**(b)** Intersection points:
\[
(0,1)
\]
and
\[
\left(\frac{b-1}{2}, \frac{b+1}{2}\right)
\]

## 9. Find all values of \(m\) such that  
\( f(x)=2x^2+x+3 \) and \( g(x)=mx-1 \) have no intersection.

Set equal:
\[
2x^2 + x + 3 = mx - 1
\]
\[
2x^2 + (1-m)x + 4 = 0
\]

No intersection means discriminant \(<0\):
\[
\Delta = (1-m)^2 - 4(2)(4) < 0
\]
\[
(1-m)^2 - 32 < 0
\]
\[
(1-m)^2 < 32
\]
\[
-4\sqrt{2} < 1-m < 4\sqrt{2}
\]

Equivalent form:
\[
1 - 4\sqrt{2} < m < 1 + 4\sqrt{2}
\]

\[
\boxed{1 - 4\sqrt{2} < m < 1 + 4\sqrt{2}}
\]
