## 1. Quadratic function \( y = x^2 - 3x - 4 \)

**(a) Complete the table**

| \(x\) | \(y\) |
|------|------|
| \(-2\) | \(6\) |
| \(0\) | \(-4\) |
| \(1\) | \(-6\) |
| \(2\) | \(-6\) |
| \(3\) | \(-4\) |
| \(5\) | \(6\) |

**(b) Intersection points with the axes**  

- \(y\)-intercept: \((0,-4)\)  
- \(x\)-intercepts: solve \(x^2 - 3x - 4 = 0\)  
  \[
  (x-4)(x+1)=0
  \]
  Points: \((4,0)\) and \((-1,0)\)

**(c) Vertex**  

\[
x_v = -\frac{b}{2a} = -\frac{-3}{2\cdot1} = \frac{3}{2}
\]

\[
y_v = \left(\frac{3}{2}\right)^2 - 3\left(\frac{3}{2}\right) - 4
= \frac{9}{4} - \frac{9}{2} - 4
= -\frac{25}{4}
\]

Vertex: \(\left(\frac{3}{2}, -\frac{25}{4}\right)\)

**(d) Concavity**  

Concave up because \(a>0\).

## 2. Quadratic function \( y = -x^2 + 2x + 3 \)

**(a) Complete the table**

| \(x\) | \(y\) |
|------|------|
| \(-1\) | \(0\) |
| \(0\) | \(3\) |
| \(1\) | \(4\) |
| \(2\) | \(3\) |
| \(3\) | \(0\) |
| \(4\) | \(-5\) |

**(b) Intersection points with the axes**  

- \(y\)-intercept: \((0,3)\)  
- \(x\)-intercepts: solve \(-x^2 + 2x + 3 = 0\)  
  \[
  x^2 - 2x - 3 = 0
  \]
  \[
  (x-3)(x+1)=0
  \]
  Points: \((3,0)\) and \((-1,0)\)

**(c) Vertex**  

\[
x_v = -\frac{b}{2a} = -\frac{2}{2(-1)} = 1
\]

\[
y_v = -(1)^2 + 2(1) + 3 = 4
\]

Vertex: \((1,4)\)

**(d) Concavity**  

Concave down because \(a<0\).

**(f) Intersection with \( y = x + 1 \)**  

\[
-x^2 + 2x + 3 = x + 1
\]

\[
-x^2 + x + 2 = 0
\]

\[
x^2 - x - 2 = 0
\]

\[
(x-2)(x+1)=0
\]

\[
x=2 \quad \text{or} \quad x=-1
\]

For \(x=2\), \(y=2+1=3\), so point \((2,3)\).  

For \(x=-1\), \(y=-1+1=0\), so point \((-1,0)\).

## 5. Find equation with one x‑intercept at \((2,0)\) and vertex \((5,-18)\)

By symmetry, the other x-intercept is \(x=8\).

\[
y = a(x-2)(x-8)
\]

Use the vertex \((5,-18)\):

\[
-18 = a(5-2)(5-8)
\]

\[
-18 = a(3)(-3) = -9a
\]

\[
a = 2
\]

\[
y = 2(x-2)(x-8)
\]

\[
y = 2(x^2 - 10x + 16)
\]

\[
y = 2x^2 - 20x + 32

## 6. Quadratic function \( y = 2x^2 + bx + c \)

**(a)** \(y\)-intercept \((0,10)\) gives \(c=10\).

**(b)** Passes through \((1,4)\):

\[
4 = 2(1)^2 + b(1) + 10
\]

\[
4 = 2 + b + 10
\]

\[
4 = b + 12
\]

\[
b = -8
\]

**(c)** Function:

\[
y = 2x^2 - 8x + 10
\]

**(d) Range**  

\[
x_v = -\frac{-8}{2\cdot2} = 2
\]

\[
y_v = 2(2)^2 - 8(2) + 10 = 2
\]

Parabola opens up, so the range is:

\[
[2,\infty)
\]

## 7. Intersection of \(f(x)=x^2-5x+4\) and \(g(x)=-x+4\)

Set equal:

\[
x^2 - 5x + 4 = -x + 4
\]

\[
x^2 - 4x = 0
\]

\[
x(x-4)=0
\]

\[
x=0 \quad \text{or} \quad x=4
\]

For \(x=0\):

\[
y = -0 + 4 = 4
\]

Point: \((0,4)\)

For \(x=4\):

\[
y = -4 + 4 = 0
\]

Point: \((4,0)\)

## 8. Functions \(f(x)=-2x^2+bx+1\) and \(g(x)=x-1\)

**(a)** One intersection at \(x=-1\):

\[
-2(-1)^2 + b(-1) + 1 = -1 - 1
\]

\[
-2 - b + 1 = -2
\]

\[
-1 - b = -2
\]

\[
b=1
\]

**(b)** With \(b=1\), solve:

\[
-2x^2 + x + 1 = x - 1
\]

\[
-2x^2 + 2 = 0
\]

\[
x^2 = 1
\]

\[
x = -1 \quad \text{or} \quad x=1
\]

For \(x=-1\):

\[
y = -1 - 1 = -2
\]

Point: \((-1,-2)\)

For \(x=1\):

\[
y = 1 - 1 = 0
\]

Point: \((1,0)\)

## 9. Find \(m\) such that \(f(x)=3x^2-x+2\) and \(g(x)=mx-4\) have **no** intersection

Set equal:

\[
3x^2 - x + 2 = mx - 4
\]

\[
3x^2 + (-1-m)x + 6 = 0
\]

No intersection means discriminant \(<0\):

\[
\Delta = (-1-m)^2 - 4\cdot3\cdot6
\]

\[
\Delta = (m+1)^2 - 72
\]

\[
(m+1)^2 - 72 < 0
\]

\[
(m+1)^2 < 72
\]

\[
-6\sqrt{2} < m+1 < 6\sqrt{2}
\]

\[
-1 - 6\sqrt{2} < m < -1 + 6\sqrt{2}
\]

Final answer:

\[
\boxed{-1 - 6\sqrt{2} < m < -1 + 6\sqrt{2}}
\]
