## 1. Quadratic function \( y = x^2 + x - 6 \)

**(a) Complete the table**

| \(x\) | \(y\) |
|------|------|
| \(-4\) | \(6\) |
| \(-2\) | \(-4\) |
| \(0\) | \(-6\) |
| \(1\) | \(-4\) |
| \(2\) | \(0\) |
| \(3\) | \(6\) |

**(b) Intersection points with the axes**  

- \(y\)-intercept: \((0,-6)\)  
- \(x\)-intercepts:  
  \[
  x^2 + x - 6 = 0
  \]
  \[
  (x+3)(x-2)=0
  \]
  Points: \((-3,0)\) and \((2,0)\)

**(c) Vertex**  
\[
x_v = -\frac{b}{2a} = -\frac{1}{2}
\]
\[
y_v = \left(-\frac{1}{2}\right)^2 - \frac{1}{2} - 6 = \frac{1}{4} - \frac{1}{2} - 6 = -\frac{25}{4}
\]
Vertex: \(\left(-\frac{1}{2}, -\frac{25}{4}\right)\)

**(d) Concavity**  
Concave up because \(a=1>0\).

## 2. Quadratic function \( y = -x^2 + 4x - 3 \)

**(a) Complete the table**

| \(x\) | \(y\) |
|------|------|
| \(0\) | \(-3\) |
| \(1\) | \(0\) |
| \(2\) | \(1\) |
| \(3\) | \(0\) |
| \(4\) | \(-3\) |
| \(5\) | \(-8\) |

**(b) Intersection points with the axes**  

- \(y\)-intercept: \((0,-3)\)  
- \(x\)-intercepts:  
  \[
  -x^2 + 4x - 3 = 0
  \]
  \[
  x^2 - 4x + 3 = 0
  \]
  \[
  (x-1)(x-3)=0
  \]
  Points: \((1,0)\) and \((3,0)\)

**(c) Vertex**  
\[
x_v = -\frac{b}{2a} = -\frac{4}{2(-1)} = 2
\]
\[
y_v = -(2)^2 + 4(2) - 3 = 1
\]
Vertex: \((2,1)\)

**(d) Concavity**  
Concave down because \(a=-1<0\).

**(f) Intersection with \( y = x + 1 \)**  
\[
-x^2 + 4x - 3 = x + 1
\]
\[
-x^2 + 3x - 4 = 0
\]
\[
x^2 - 3x + 4 = 0
\]
Discriminant:
\[
\Delta = (-3)^2 - 4(1)(4) = 9 - 16 = -7
\]
No real intersection points.

## 5. Find the equation of a quadratic with one \(x\)-intercept at \((1,0)\) and vertex \((3,-8)\)

By symmetry, the other \(x\)-intercept is \(x=5\).

\[
y = a(x-1)(x-5)
\]

Use the vertex \((3,-8)\):
\[
-8 = a(3-1)(3-5) = a(2)(-2) = -4a
\]
\[
a=2
\]

\[
y = 2(x-1)(x-5)
\]
\[
y = 2(x^2 - 6x + 5) = 2x^2 - 12x + 10
\]

## 6. Quadratic function \( y = 2x^2 + bx + c \)

**(a)** \(y\)-intercept \((0,-4)\) gives  
\[
c=-4
\]

**(b)** Passes through \((1,3)\):
\[
3 = 2(1)^2 + b(1) - 4
\]
\[
3 = 2 + b - 4
\]
\[
3 = b - 2
\]
\[
b=5
\]

**(c)** Full equation:
\[
y = 2x^2 + 5x - 4
\]

**(d) Range**  
\[
x_v = -\frac{5}{4}
\]
\[
y_v = 2\left(-\frac{5}{4}\right)^2 + 5\left(-\frac{5}{4}\right) - 4
= \frac{25}{8} - \frac{25}{4} - 4
= -\frac{57}{8}
\]
Range:
\[
\left[-\frac{57}{8}, \infty\right)
\]

## 7. Find the intersection points of  
\( f(x) = 2x^2 - 3x - 2 \) and \( g(x) = x - 2 \)

Set equal:
\[
2x^2 - 3x - 2 = x - 2
\]
\[
2x^2 - 4x = 0
\]
\[
2x(x-2)=0
\]
\[
x=0 \quad \text{or} \quad x=2
\]

For \(x=0\):
\[
y = 0 - 2 = -2
\]
Point: \((0,-2)\)

For \(x=2\):
\[
y = 2 - 2 = 0
\]
Point: \((2,0)\)

## 8. Functions \( f(x) = -x^2 + bx + 2 \) and \( g(x) = x + 2 \)

**(a)** One intersection occurs at \(x=0\), so
\[
f(0)=g(0)
\]
\[
2=2
\]
This is true for any \(b\), so \(b\) is not fixed by part (a).

To make the question solvable with two intersections, use the equation:
\[
-x^2 + bx + 2 = x + 2
\]
\[
-x^2 + (b-1)x = 0
\]
\[
x[-x + (b-1)] = 0
\]
Thus one solution is always \(x=0\), and the other is
\[
x = b-1
\]

**(b)** The intersection points are
\[
(0,2)
\]
and
\[
(b-1, b+1)
\]

## 9. Find all values of \(m\) such that  
\( f(x) = x^2 - 4x + 5 \) and \( g(x) = mx + 1 \) have no intersection.

Set equal:
\[
x^2 - 4x + 5 = mx + 1
\]
\[
x^2 + (-4-m)x + 4 = 0
\]

No intersection means discriminant \(<0\):
\[
\Delta = (-4-m)^2 - 4(1)(4) < 0
\]
\[
(m+4)^2 - 16 < 0
\]
\[
(m+4)^2 < 16
\]
\[
-4 < m+4 < 4
\]
\[
-8 < m < 0
\]

\[
\boxed{-8 < m < 0}
\]
