# IB Extended Maths

This repository stores IB Extended Maths teaching, practice, and reference material. The folders are organized by topic and by resource type so it is easy to find the right lesson, worksheet, or archive file.

## Suggested Workflow

Start with the topic folder that matches the unit you are teaching or revising, then open the lesson or practice set inside that folder. Use the archive and general folders when you need reference material, syllabus documents, or older assessment examples.
